### API Usage

```typescript
const goplusResult = await goplus.getTokenSecurity({
  chainId: '1',
  contractAddress: '0x...'
});

// Calculate risk based on factors
let risk = 0;
if (goplusResult.is_honeypot) risk += 50;
if (parseFloat(goplusResult.sell_tax) > 10) risk += 30;
if (goplusResult.hidden_owner) risk += 20;
```

## Shieldeum Integration

Shieldeum provides decentralized threat intelligence:

```typescript
interface ShieldeumThreatFeed {
  // Known malicious addresses
  blacklist: {
    address: string;
    chain: string;
    category: 'phishing' | 'scam' | 'exploit' | 'mixer';
    reportedAt: string;
    confidence: number;
  }[];
  
  // Known safe addresses
  whitelist: {
    address: string;
    chain: string;
    category: 'exchange' | 'protocol' | 'dao';
    verifiedAt: string;
  }[];
  
  // Real-time alerts
  alerts: {
    type: 'new_exploit' | 'rug_pull' | 'phishing_campaign';
    severity: 'low' | 'medium' | 'high' | 'critical';
    details: string;
    timestamp: string;
  }[];
}
```

### Real-Time Feed

```typescript
// Subscribe to threat feed
const shieldeum = new ShieldeumClient({
  apiKey: process.env.SHIELDEUM_API_KEY
});

shieldeum.subscribe('threats', (threat) => {
  // Update local blacklist
  localBlacklist.add(threat.address);
  
  // Alert affected users
  if (threat.severity === 'critical') {
    notifyUsers(threat);
  }
});
```

## Data Aggregation

### Priority Hierarchy

When multiple sources disagree, CryptoGuard uses weighted aggregation:

| Source | Weight | Specialty |
|--------|--------|-----------|
| CryptoGuard Internal | 35% | Web3-specific ML models |
| CUBE3.AI | 25% | Transaction simulation |
| GoPlus | 20% | Token security |
| EtherScan/BSCScan | 15% | Contract verification |
| Shieldeum | 5% | Community reports |

### Conflict Resolution

```typescript
function aggregateRiskScores(sources: RiskSource[]): number {
  const weightedSum = sources.reduce((sum, source) => {
    return sum + (source.score * source.weight * source.confidence);
  }, 0);
  
  const totalWeight = sources.reduce((sum, source) => {
    return sum + (source.weight * source.confidence);
  }, 0);
  
  return weightedSum / totalWeight;
}
```
