## Data We Collect

### Extension Analytics (Optional)

```typescript
interface AnonymousAnalytics {
  // Aggregated usage metrics (no PII)
  sessionCount: number;
  featuresUsed: string[];
  errorsEncountered: string[];
  
  // Performance metrics
  avgScanTime: number;
  extensionVersion: string;
  browserType: string;
  
  // NOT collected
  walletAddresses: never;
  transactionHistory: never;
  browsingHistory: never;
  personalInfo: never;
}
```

<Note>
  Analytics are opt-in and completely anonymous. No personal information is ever collected.
</Note>

### URL Scanning Data

When scanning URLs, we process:

| Data | Purpose | Retained |
|------|---------|----------|
| URL | Threat analysis | 24 hours (hashed) |
| Scan Result | Improve detection | Aggregated only |
| Timestamp | Rate limiting | 24 hours |

**NOT Processed:**
- Page content (analyzed locally only)
- Form inputs
- Cookies or session data
- Login credentials

### Transaction Analysis Data

```typescript
interface TransactionScanData {
  // Processed for security analysis
  contractAddress: string;      // Public blockchain data
  functionSelector: string;     // Public transaction data
  chainId: number;
  
  // NOT processed
  privateKey: never;
  signedTransaction: never;
  personalIdentifiers: never;
}
```

## Data Storage

### Local Storage (Your Device)

```typescript
interface LocalData {
  // User preferences
  settings: UserSettings;
  whitelist: string[];
  
  // Cached analysis results
  recentScans: CachedScan[];
  
  // Session data
  sessionToken: string;  // Encrypted
}
```

### Server Storage

| Data Type | Location | Encryption | Retention |
|-----------|----------|------------|-----------|
| API Keys | HSM (Hardware Security Module) | AES-256-GCM | Until revoked |
| Anonymous Analytics | AWS (US-East) | AES-256 | 90 days |
| Threat Signatures | Global CDN | N/A (public) | Indefinite |

## Data NOT Collected

CryptoGuard explicitly does NOT collect or transmit:

- ❌ Private keys or seed phrases
- ❌ Wallet balances or holdings
- ❌ Transaction history
- ❌ Browsing history beyond current scan
- ❌ Personal identification information
- ❌ IP addresses (hashed for rate limiting only)
- ❌ Cookies from websites you visit
- ❌ Form data or passwords
- ❌ Email addresses (unless you contact support)

## Third-Party Data Sharing

### Security Partners

We share anonymized, aggregated threat data with:

| Partner | Data Shared | Purpose |
|---------|-------------|---------|
| CUBE3.AI | Transaction patterns | Fraud detection |
| GoPlus | Contract addresses | Security analysis |
| Shieldeum | Threat signatures | Community protection |

**No personal data is ever shared with third parties.**

### Law Enforcement

We may disclose information if required by law:

- Valid court orders
- Subpoenas
- Legal process

In such cases, we can only provide:
- Aggregated, anonymized statistics
- No personal user data (we don't have it)

## Your Rights

### GDPR Compliance

EU users have the right to:

| Right | Implementation |
|-------|----------------|
| Access | Export your local data via extension settings |
| Erasure | Uninstall extension removes all local data |
| Portability | Export settings in JSON format |
| Objection | Opt-out of analytics at any time |

### CCPA Compliance

California users have the right to:

- Know what data is collected (see above)
- Delete personal information (N/A - we don't collect PII)
- Opt-out of sale (we never sell data)
