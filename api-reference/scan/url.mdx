---
title: "Scan URL"
api: "POST https://api.cryptoguard.ai/v1/scan/url"
description: "Scan a URL for phishing, malware, and security threats"
---

Analyze any URL for potential security threats including phishing, malware, and suspicious content.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer cg_live_xxxxxxxx`
</ParamField>

## Body Parameters

<ParamField body="url" type="string" required>
  The URL to scan. Must be a valid HTTP/HTTPS URL.
  
  **Example**: `https://suspicious-site.com/login`
</ParamField>

<ParamField body="chain" type="string" default="ethereum">
  Blockchain context for dApp analysis.
  
  **Options**: `ethereum`, `bsc`, `polygon`
</ParamField>

<ParamField body="options" type="object">
  Scan configuration options.
  
  <Expandable title="options properties">
    <ParamField body="options.includeContent" type="boolean" default="false">
      Include page content analysis (increases latency)
    </ParamField>
    
    <ParamField body="options.includeWhois" type="boolean" default="true">
      Include domain WHOIS information
    </ParamField>
    
    <ParamField body="options.timeout" type="integer" default="5000">
      Maximum time to wait for page load (ms). Range: 1000-30000
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="url" type="string">
  The scanned URL (may differ from input due to redirects)
</ResponseField>

<ResponseField name="riskScore" type="integer">
  Overall risk score from 0-100. Higher = more risky.
</ResponseField>

<ResponseField name="riskLevel" type="string">
  Risk classification: `safe`, `low`, `medium`, `high`, `critical`
</ResponseField>

<ResponseField name="threats" type="array">
  List of detected threat types.
  
  **Possible values**: `phishing`, `malware`, `scam`, `suspicious`, `new_domain`, `typosquat`, `invalid_ssl`
</ResponseField>

<ResponseField name="details" type="object">
  Detailed scan results.
  
  <Expandable title="details properties">
    <ResponseField name="details.domainAge" type="integer">
      Domain age in days
    </ResponseField>
    
    <ResponseField name="details.sslStatus" type="string">
      SSL certificate status: `valid`, `invalid`, `expired`, `self_signed`, `missing`
    </ResponseField>
    
    <ResponseField name="details.redirectCount" type="integer">
      Number of redirects followed
    </ResponseField>
    
    <ResponseField name="details.finalUrl" type="string">
      Final URL after redirects
    </ResponseField>
    
    <ResponseField name="details.ipAddress" type="string">
      Resolved IP address
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="recommendation" type="string">
  Action recommendation: `PROCEED`, `CAUTION`, `AVOID`, `BLOCK`
</ResponseField>

  }'

const response = await fetch('https://api.cryptoguard.ai/v1/scan/url', {
  method: 'POST',

```python Python
import requests

response = requests.post(
    'https://api.cryptoguard.ai/v1/scan/url',
    headers={
        'Authorization': f'Bearer {api_key}',
        'Content-Type': 'application/json'
    },
    json={
        'url': 'https://app.uniswap.org',
        'chain': 'ethereum',
        'options': {
            'includeContent': True,
            'includeWhois': True
        }
    }
)

data = response.json()
print(data['data']['riskScore'])
```
